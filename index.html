Objective 1: Changing an element's CSS class using JavaScript

An HTML element is made up of two HTML tags that enclose inner content.

If you navigate to example.com, the text "Example Domain" is contained within a h1 element. See for yourself, by right clicking on that text, to open a context menu, and selecting the option Inspect.

In JavaScript, there is a special word, document that references the entire HTML page on the screen. It is a variable that represents our HTML page. This variable is also refered to as the Document Object Model or "DOM" for short. Using the DOM we can affect the HTML document (webpage) we are viewing.

Let's use the DOM to help us select something on the webpage. Let's catch one of the Pokémon below! They've each been assigned an HTML id that corresponds to their pokémon name. Given an id is unique, it is a good identifier to target something based on. The way we can do this is by using one of the DOM's built-in methods, getElementById.

Now, let's catch some Pokémon!

In the JavaScript console, let's select Charmander and remove him (put him into his pokéball). See what the code below does by clicking on the "Execute" button.

        var charmander = document.getElementById("charmander");
charmander.className = "caught";

Note: refreshing the page sets it to its original state, making Charmander reappear.

How does this code make Charmander disappear? We targeted the Charmander img element and added the class, caught. This class has the CSS rule opacity: 0 applied to it, making the element transparent.

Objective 2: Capturing User Input

We've just learned how we can capture Pokémon by writing JavaScript that manipulates our HTML. Let's now learn how to capture a user's input in JavaScript using a prompt.

Great! Now let's prompt the user for some input and see the result displayed in the console.

        var sentiment = prompt("How are you feeling today?");
console.log("Today you are", sentiment);

Note: console.log() is just a simple way to print something to the console.

Try clicking the above button. It should produce a prompt message that will display what it's input to the console as output.

Objective 3: User Output

Now that we can get user input and select elements on a page, we can think about how to display a user's input back to them! We'll have to change what's inside of an HTML element first by selecting it, then setting its innerHTML to anything we want. Let's do this to update the text below in order to reflect something we've learned today.

Today I learned: ...

The elipse above is wrapped in an HTML span tag that has the id "new-knowledge". Knowing this, we should be able to easily select it. Thereafter, we can change what's inside of it to reflect something we've learned. Let's see an example of how this will work.

            var newKnowledge = document.getElementById("new-knowledge");
newKnowledge.innerHTML = "how to select and manipulate DOM elements!"
